## 已完成

### v1.0 — 核心系統上線（2026/06/17）

**前台：**
- 套餐組 A～H
- 單點區（東坡肉、醬燒蹄膀、醬燒豬腳、香滷嫩筍）
- 取貨方式：自取 / 低溫宅配 / 7-11店到店
- 付款方式：現金（自取）/ 匯款（全部）
- 匯款資訊顯示
- 運費自動計算（滿$3000免運 / 宅配$200 / 7-11 $150）
- 訂單確認摘要

**後台：**
- 訂單列表（依取貨方式分區）
- 篩選頁籤
- 付款/出貨狀態修改
- 已完成邏輯（已出貨/自取完成移入已完成區）
- 統計數字
- 60秒自動重整
- 後台密碼保護（`zaoshi2026`）

**基礎建設：**
- Google Sheet 三個分頁（訂單、商品設定、系統設定）
- Apps Script Web App 串接
- GitHub Pages 部署（mumuworks/zaoshi-order）
- LOGO 去背嵌入

## 開發原則

1. **不重構**：每次只針對特定功能修改，不動已正常運作的部分
2. **MVP 優先**：先讓功能可用，再優化體驗
3. **手機優先**：所有 UI 以手機操作為主要考量
4. **低成本**：優先使用免費工具，確定長期需求再投入費用
5. **可維護**：程式碼保持可讀，重要設定集中在檔案頂部

## 版本命名規則

```
v主版本.次版本.修補版本

主版本：重大功能或架構改變
次版本：新增功能
修補版本：Bug 修正或小調整

範例：
v1.0   → 初次上線
v1.1   → 新增自取時段
v1.3.1 → 修補圖片功能（1.3的子版本）
```
